# 主题变了的影响范围

```js
@primary-color: #002766; // 全局主色: Button
@link-color: #002766; // 链接色: 链接类型的按钮
@success-color: #092b00; // 成功色: alert 组件
@warning-color: #613400; // 警告色: alert 组件
@error-color: #5c0011; // 错误色: 表单校验失败
@font-size-base: 16px; // 主字号
@heading-color: rgba(0, 0, 0, 0.45); // 标题色: alert 组件 text 内容
@text-color: rgba(0, 0, 0, 0.35); // 主文本色: form label color
@text-color-secondary: rgba(0, 0, 0, 0.25); // 次文本色: button default
@disabled-color: rgba(0, 0, 0, 0.15); // 失效色: disabled button
@border-radius-base: 4px; // 组件/浮层圆角: 所有表单组件基础圆角
@border-color-base: #8c8c8c; // 边框色: 所有按钮的边框
@box-shadow-base: 0 4px 8px -4px rgba(0, 0, 0, 0.22); // 浮层阴影: 默认的阴影
```
